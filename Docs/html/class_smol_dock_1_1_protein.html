bool SmolDock::Protein::populateFromPDB ( const std::string &  filename)

For reference purpose, the hierarchy of the PDB file reading is : System (defined by ESBTL, like input filter in some ways) Model (PDB-file specification : more or less, protein) Chain (Amino acid chain) Residue (=Amino acid) Atom

Heteroatoms (Cl, or crystallographic ions) are mixed in at the residue level. We detect those if residueName is not in {set of defined residueName} TODO : Differentiate between legit biological ions (metalloprotein, ...) and crystallographic heavy atoms

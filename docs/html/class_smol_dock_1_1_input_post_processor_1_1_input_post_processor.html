Detailed Description

Interface definition for InputPostProcessor.

The purpose of input post processor is mainly to modify ligand and protein atom & residue to simulate the behaviour of a particular software, and get numerically identical results. Though other uses are possible and imaginable.

For example, we consider cysteine sulfur atom as a H-bond acceptor, and sometime donor (if not part of a disulfide bridge). However, Vina does not. (Because X-Score does not either). So we correct that in a post processing step.

We find that such differences are often rather minimal and do not warrant custom importers.
